## Install

```
npm i @visualjs/grid
```

or using yarn

```
yarn add @visualjs/grid
```

If you want to use the latest beta version, please specify the @next tag when installing.

```
npm i @visualjs/grid@next
```

or using yarn

```
yarn add @visualjs/grid@next
```
